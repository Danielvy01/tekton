# TEKTON - Actividad

Este es un challenge para TEKTON

## Description

Este challenge contiene las mejores prácticas de la programación 2024

1.1. API Rest en .net Core (última versión).

1.2. Web Api Documentada usando swagger.

1.3. Uso de patrones (mediator pattern, repository pattern, cqrs, etc).

1.4. Aplicando principios SOLID y Clean Code.

1.5. Implementando la solución haciendo uso de TDD.

1.6. Buenos patrones para las validaciones del Request, y además,
    considera los HTTP Status Codes en cada petición realizada.

1.7. Estructurar el proyecto en N-capas orientado a microservicios

## Getting Started

### Dependencies

Visual Studio 2022

SqlServer

REDIS

DockerDesktop (*Opcional, pero necesario en caso no tenga un rediscache)



El precio del descuento para el api GetById está en el siguiente enlace:
https://6672fa246ca902ae11b2a520.mockapi.io/api/v1/discount

Ejecutar el siguiente script para crear la base de datos:
script_createdatabase.sql

### Installing

Para ejecutar el proyecto en dockerDesktop ejecutar el siguiente comando:

docker-compose -f docker-compose.yml up -d --force-recreate --build -d

### Executing program

*EL LOG va aparecer en la carpeta C:\LOG\
